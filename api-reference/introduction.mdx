  This API documentation provides endpoints for the Kuralynx automated calling platform.
  You can use these endpoints to initiate calls through your configured agents.
</Note>

## Welcome

The Kuralynx API allows you to integrate automated calling capabilities into your applications. This API provides endpoints for initiating calls using pre-configured agents.

## Base URL

All API requests should be made to:
```
https://api.kuralynx.com/v1
```

## Getting Started

1. **Obtain your API key** from your Kuralynx dashboard
2. **Configure your agents** in the Kuralynx platform
3. **Use the API endpoints** to initiate calls programmatically
